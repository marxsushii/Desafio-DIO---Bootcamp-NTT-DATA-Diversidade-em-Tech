História dos computadores

A história dos computadores pode ser divididas por gerações. Os primeiros computadores ocupavam grandes salas e se utilizavam de muita energia para manter seu funcionamento. Grandes nomes de destacaram na primeira geração como Mark I, ENIAC e Colossus. A segunda geração por sua vez foi marcada por computadores que utilizavam-se de transistores para o funcionamento. Os demais gerações foram marcadas pelo surgimento de computadores cada vez menores e populares, de maneira que tais computadores passaram a se popularizar entre empresas e também como parte da casa das pessoas.

O que são Clients?

Clients ou Clientes são aqueles que ficam na extremidade do usuário em uma rede. São muitas vezes computadores ou dispositivos mobile que fazem o acesso a aplicações por meio da internet. essas aplicações são por vezes disponibilizadas através da internet e se utilizam de arquivos estáticos para tal. Um Client cria uma requisição para um Server, esse que por sua vez retorna a requisição com os dados solicitados, o Client a partir disso realiza a leitura e formatação dos dados para o usuário.

O que são Servers?

Os Serves ou Servidores são aqueles que pertencem a outra extremidade da rede, são aqueles que disponibilizam informações para as aplicações que são executadas no Client. Utilizam-se muitas vezes de linguagens específicas para tal como por exemplo Java e C++. O Javascript é uma linguagem de programação que possui flexibilidade para ser executada no client side e também no server side via NodeJS, trazendo dessa maneira muita versatilidade para a cominicação entre o FrontEnd e o BackEnd.

Como é uma estrutura básica de um HTML?

O HTML ou Hypertext Markup Language possui uma estrutura básica definida pelas tags html, head e body. Ao definir essas tags já é possível inicializar a execução do HTML no navegador de sua preferência

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas são definidas pelas tags ol e listas não ordenadas são definidas pelas tags ul. Os componentes de listas ordenadas ou não ordenadas são definidos por sua vez pela tag li.

Resumo Tags Aprendidas

    >
  • h1 até h6 Tags utilizadas para definir títulos, seus tamanhos são diferentes e vão de h1, como o maior tamanho, até h6, como o menor tamanho.
  • p
    • Tag utilizada para definir parágrafos de texto.
  • mark
    • A tag mark define um trecho de textaque em um texto. Traz para o destque o efeito de 'marca texto'.
  • small e strong
    • A tag small define um ajuste num tamanho do texto, tornando-o menor do que o padrão da tag no qual foi incluído. A tag strong por sua vez define o engrossamento da fonte utilizada, dando destaque para o texto. Tem um efeito parecido com alterações que são utilizadas pela tag b ou ainda pelo ajuste de font-weight para valores maiores.
  • 'i' e 'u'
    • As tags i e u definem respectivamente alterações a serem cplicadas na fonte para tranforma-las em formato itálico e sublinhado.
  • ol, ul e li
    • As tags ol e ul, respectivamente, definem se uma lista será ordenada ou não ordenada. Os componentes da lista, seja ela ordenada ou não, deverão ser escritos nas tags li.
  • 'a'
    • Tag utilizada para criar links, sejam eles externos ou internos (por meio de ancoragem). Para se utilizar o conceito de ancoragem, deve-se definir href como a id da tag desejada para realizar a ancoragem.
  • hr
    • A tag hr é utilizada para definir uma linha horizontal. Essa linha pode ser utilizada como divisor, um elemento visual, para auxiliar na leitura de uma página. Por ser uma tag sem conteúdo, pode ser aberta e fechada sem a criação de tags separadas.
  • sub e sup
    • As tags sub e sup costumam ser utilizadas para fórmulas matemáticas e/ou quimicas pois definem posicionamento do caracter em relação a tag adjascente.
  • blockquote
    • A tag blockquote define uma longa citação.
  • font
    • Atualmente a tag font não possui mais uso recomendado, apesar disso ela possui a capacidade de definir ajustes na fonte utilizada.
  • del
    • Demarca uma parte que foi teletada do texto original. Pode ser utilizada para manter o histórico de alterações de um texto.
  • abbr
    • A tag abbr define uma abreviação de um texto com base nas especificações criadas
